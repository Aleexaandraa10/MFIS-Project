# Festival Management Platform

This project is a **Java-based festival management system** developed for the PAO course. It simulates a real-world festival ecosystem (similar to Untold or Neversea), including participants, events, tickets, and organizers, using a modular architecture with DAO, Service layers, and a relational database.

## Core Functionality

The application supports core operations such as:

- Managing participants, events, tickets, and organizers  
- Handling multiple event types (Concert, DJ, CampEats, GlobalTalks, FunZone)  
- Seat reservations for limited-capacity events  
- A points and rewards system  
- Audit logging of user actions  
- CLI-based interaction with validation and business logic  

---

## Purpose in Current Context (MFIS)

This project will be used as a **starting point (reference application)** for testing and validating the **AVMF framework** within the MFIS course.

It provides:

- A sufficiently complex domain  
- Multiple architectural layers (DAO, Service, Model)  
- Realistic use cases and workflows
